## How to use

### Browser

```js
<script src="lib/Matrix2D.js">
<script>
console.log( Matrix2D.identity() );
</script>
```

### WebWorkers

```js
importScripts("lib/Matrix2D.js");
console.log( Matrix2D.identity() );
```

### Node.js

```js
var Matrix2D = require("lib/Matrix2D.js");
console.log( Matrix2D.identity() );
```
